# Alacritty Themes

## Available Themes

This directory contains the following color schemes:

- **blood_moon.toml** - Dark theme with red accents
- **catppuccin_mocha.toml** - Catppuccin Mocha color scheme
- **cobalt2.toml** - Cobalt2 theme

## Using Themes

To use a theme, import it in your `alacritty.toml`:

```toml
[general]
import = ["~/.config/alacritty/themes/catppuccin_mocha.toml"]
```

Or using the absolute path from this dotfiles repo:

```toml
[general]
import = ["~/dotfiles/config/alacritty/themes/catppuccin_mocha.toml"]
```

## More Themes

See the [alacritty-theme repository](https://github.com/alacritty/alacritty-theme) for additional themes.
